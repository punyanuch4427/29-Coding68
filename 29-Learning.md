ขั้นตอนการเขียนโปรแกรม
1.รับข้อมูลน้ำหนักุ
2.คำนวณหาปริมาณน้ำ และหาว่าปริมาณน้ำเท่านี้จะต้องกินน้ำกี่ขวด ขวดละกี่ลิตร
input : ข้อมูลน้ำหนักุ
process : คำนวณหาปริมาณน้ำ
output : บอกปริมาณน้ำเท่านี้จะต้องกินน้ำกี่ขวด ขวดละกี่ลิตร
1.ไม่ยากเกินไป
2.รหัสจำลอง เพราะว่าสามารถเห็นขั้นตอนได้อย่างละเอียด ทำให้เกิดความเข้าใจในโปรแกรม
3.ยากที่สุด คือ ขั้นตอนการคิดปริมาณขวดน้ำที่ต้องดื่ม แก้ไขโดย ศึกษาวิธีการให้ดีเพื่อให้เข้าใจขั้นตอนมากขึ้น
